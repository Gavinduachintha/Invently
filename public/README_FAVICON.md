# Invently Favicon

This folder contains the favicon and icon files for the Invently application.

## Files:

- `invently-favicon.svg` - Main favicon (32x32) used in browser tabs
- `invently-icon.svg` - High-resolution icon (512x512) for PWA and app icons
- `manifest.json` - Web app manifest for Progressive Web App functionality

## Colors Used:

- Primary: #10b981 (Emerald 500)
- Secondary: #14b8a6 (Teal 500)
- Accent: #059669 (Emerald 600)

## Design:

The favicon features a stylized "I" letter on a gradient emerald-to-teal background, matching the Invently brand identity.
